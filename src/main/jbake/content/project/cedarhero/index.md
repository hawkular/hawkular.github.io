ingenieux cedarhero is our set of [Apache Maven](http://maven.apache.org/) plugins for [Heroku](http://www.heroku.com), which lets you effectively use Maven as a deployer without messing around your pom too much.

### Getting Started

In order to effectively use cedarhero, you need to have a basic understanding of [Apache Maven](http://maven.apache.org/) and [Heroku](http://www.heroku.com/).
